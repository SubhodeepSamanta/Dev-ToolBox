## üöÄ Quick Start

### Prerequisites

- Node.js (v18+)
- npm (v9+)
- MongoDB 
- Git
- Zustand
- Tailwind CSS

### 1. Clone the Repository

```bash
git clone https://github.com/SubhodeepSamanta/Dev-ToolBox.git
cd Dev-ToolBox
```

### 2. Install Dependencies

```bash
cd Frontend
npm install
```

### 3. Frontend Setup

```bash
cd ../Frontend
npm install

# Create .env file and add:
VITE_API_ENDPOINT=http://localhost:5000
```

### 4. Run the App

```bash
# Backend
cd Backend
npm run dev

# Frontend (new terminal)
cd ../Frontend
npm run dev
```
